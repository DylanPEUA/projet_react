
# Portfolio React

Ce projet est un portfolio web développé avec React, illustrant des compétences, des réalisations et des services proposés. Il inclut une navigation responsive, une modale de profil, et un formulaire de contact.

## Fonctionnalités principales

- Accueil, services, portfolio, contact, mentions légales
- Navigation responsive avec menu mobile
- Effets visuels sur les liens et les cards
- Footer avec liens et réseaux sociaux
- Modale d'affichage de profil

## Prérequis

- Node.js >= 16.x
- npm >= 8.x

## Installation

1. Clonez le dépôt ou téléchargez le projet.
2. Placez-vous dans le dossier du projet :
	```bash
	cd projet_react
	```
3. Installez les dépendances :
	```bash
	npm install
	```

## Lancement du projet

Pour démarrer le serveur de développement :

```bash
npm start
```

Le site sera accessible sur [http://localhost:3000](http://localhost:3000).

## Construction pour la production

```bash
npm run build
```

## Tests

```bash
npm test
```

## Technologies utilisées

- React
- React Router
- Tailwind CSS
- React Icons
